# 我的个人网站

这是一个使用 GitHub Pages 搭建的个人网站。

## 使用说明

### 设置 GitHub Pages

1. 进入仓库的 Settings（设置）
2. 在左侧菜单中找到 "Pages"
3. 在 "Build and deployment" 下：
   - Source: 选择 "GitHub Actions"
4. 保存设置

### 本地预览

直接在浏览器中打开 `index.html` 文件即可预览网站。

### 自定义内容

编辑以下文件来自定义你的网站：

- `index.html` - 网站的主要内容
- `style.css` - 网站的样式设计

### 部署

当你推送代码到 `main` 分支时，GitHub Actions 会自动部署你的网站到 GitHub Pages。

部署完成后，你可以通过以下地址访问你的网站：
`https://runsqrun.github.io/mysite_test/`

## 特性

- 响应式设计，支持移动设备
- 现代化的界面
- 简单易用，容易定制
